# Timeline

A Timeline component contains [Timeline.Item](./Item.md) child components, and presents them with a timeline-UI.


## Example

```jsx
<Timeline>
  <Timeline.Item>
    ...
  </Timeline.Item>
  <Timeline.Item>
    ...
  </Timeline.Item>
</Timeline>
```


## Props

| Prop | Type | Description |
| --- | --- | --- |
| className | string | Custom class names to be added to the component. |
